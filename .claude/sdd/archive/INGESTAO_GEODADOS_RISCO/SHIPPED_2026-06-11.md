de retornar `score_partial=True` para 100% dos imóveis.

## Lições Aprendidas

### Processo
- **DEFINE completo elimina surpresas no BUILD**: A especificação das 7 fontes com URLs exatas,
  estratégia de upsert e casos de borda (código 6→7 dígitos CEMADEN) no DEFINE fez com que o
  BUILD seguisse sem iterações inesperadas.
- **`MeshResult` como intermediário explícito é melhor que estado global**: Passar o mesh dict
  como argumento para `CemadenLoader.load(mesh=...)` torna a dependência explícita e testável,
  em vez de carregar o shapefile duas vezes ou usar um cache global.

### Técnico
- **geopandas não suporta leitura direta de `zipfile.ZipFile` aberto**: `gpd.read_file()` com
  um handle de arquivo dentro de um ZIP funciona em versões mais recentes, mas o método mais
  portável e robusto é extrair os sidecar files (`.dbf`, `.prj`, `.shx`) para um `tempfile.TemporaryDirectory`
  e então chamar `gpd.read_file(path)` com caminho absoluto.
- **`FunaiLoader` deve propagar HTTP errors, não silenciar**: Diferentemente de `CemadenLoader`
  (que tem URLs de fallback), `FunaiLoader` tem uma única URL de produção. Propagar a exceção
  ao job-level handler (que loga e marca `partial_layers`) é mais correto do que retornar `errors[]`
  vazio — o teste original tentava assertar `stats` em uma chamada que levantava exceção.
- **COALESCE no UPDATE de `ibge_municipality_stats` é essencial para rodadas incrementais**:
  Cada loader atualiza apenas as colunas que conhece (ex: `AtlasBrasilLoader` só sabe IDH).
  Sem `COALESCE(EXCLUDED.col, target.col)`, uma rodada que só atualiza IDH zeraria
  `population_2022` e `homicide_rate` já populados.
- **Colunas do Atlas Brasil XLS são instáveis entre edições**: A detecção fuzzy com `_find_col()`
  (lista de candidatos + busca por substring case-insensitive) é mais robusta que hardcode.
  Mesmo padrão aplicado ao IPEA para o nome da coluna de taxa de homicídio.
- **Ruff F841 dentro de `with zipfile.ZipFile` é sutil**: Atribuição `shp_bytes = zf.read(name)`
  foi copiada do DESIGN como rascunho antes de perceber que o `read()` não era necessário com
  a abordagem de extração para tempdir. Ruff detectou corretamente.

### Ferramentas
- **`shapely.to_wkt()` é mais rápido que `.wkt` property**: Para batches grandes de geometrias,
  `to_wkt(geom)` usa a API C diretamente; `.wkt` passa por Python puro. Diferença mensurável
  com 1000+ polígonos do ICMBio.
